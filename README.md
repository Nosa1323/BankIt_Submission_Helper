# BankIt_Submission_Helper
This script helps to change the format of the features for BankIt submission.

**The aim of this script** is transforming data into BankIt [acceptable format](https://www.ncbi.nlm.nih.gov/WebSub/html/help/feature-table.html): five-column table. The script turning data into acceptable format according to specific rules described in details [here](https://www.insdc.org/documents/feature_table.html#5.2).
